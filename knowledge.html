Web Portfolio ตัวอย่าง

Web Portfolio นี้ใช้เทคนิคการออกแบบที่เน้นความมั่นคง (สีหลัก) และการกระตุ้นการโต้ตอบ (Dynamic CTA) เพื่อนำเสนอความเชี่ยวชาญด้านวิศวกรรมไฟฟ้าได้อย่างทันสมัย

เยี่ยมชมเว็บไซต์จริง

1. แนวคิดการออกแบบและการจัดการสี

Dynamic Color Consistency

  • กำหนดสีธีมหลัก (น้ำเงิน/ฟ้า) เพื่อสื่อถึงความเป็นมืออาชีพและความน่าเชื่อถือ
  • ใช้ **Dynamic Gradient** (ม่วง/น้ำเงิน) สำหรับปุ่ม CTA หลัก (`.btn-hire`, `.app-link-btn`) เพื่อสร้างพลังงานและกระตุ้นการคลิกเข้าชม
  • การใช้ตัวแปร CSS (`:root`) ช่วยให้คุณเปลี่ยนสีทั้งเว็บได้ง่ายและรวดเร็ว
/* CSS Variables Example */ :root { --dynamic-blue: #2c68f7; --dynamic-gradient: linear-gradient(90deg, var(--dynamic-blue) 0%, var(--dynamic-purple) 100%); }

2. การจัดการรูปทรงรูปโปรไฟล์ (Mobile-Friendly)

เพื่อความเสถียรและสุนทรียภาพที่ดีบนทุกหน้าจอ เราได้เปลี่ยนไปใช้การจัดการรูปทรงที่แยกตามขนาดหน้าจอ:

Desktop/Tablet (สี่เหลี่ยมผืนผ้าขอบมน)

  • ใช้ภาพสี่เหลี่ยมผืนผ้าแบบ Portrait (อัตราส่วน 4/5) เพื่อความเป็นมืออาชีพ
  • ไม่มีพื้นหลังสีมาซ้อน โดยเน้นความโดดเด่นด้วย `box-shadow` ที่ตัวภาพเอง

Mobile (วงกลมสมมาตร)

  • ใช้ Media Query บังคับให้ Container ของรูปภาพเป็นวงกลม (`border-radius: 50%`) บนหน้าจอเล็ก (ต่ำกว่า 600px)
  • เทคนิคนี้ทำให้รูปภาพถูกตัดเป็นวงกลมโดยอัตโนมัติ ทำให้เว็บดูทันสมัยและประหยัดพื้นที่บนมือถือ
**โค้ดสำหรับ Mobile (วงกลม):**
@media (max-width: 600px) { .hero-image-container { width: 200px; height: 200px; border-radius: 50%; /* ทำให้ Container เป็นวงกลม */ } .profile-image { width: 100%; height: 100%; /* ภาพจะถูกตัดตามขอบ Container */ } }

3. เทคนิคเพื่อความรื่นไหล (Smooth Flow)

Smooth Image Transition (Fade In/Out)

  • ใช้ CSS `transition: opacity 0.5s ease-in-out` ใน `.profile-image` ทำให้การเปลี่ยนภาพเป็นไปอย่างนุ่มนวล
  • JavaScript สลับ `opacity` เป็น 0 ก่อนเปลี่ยน `src` และเปลี่ยนกลับเป็น 1 เพื่อให้เกิด Effect Fade

Typewriter Effect (ตัวอย่างด้านล่าง)

เพิ่มความเป็น Dynamic ด้วย Typewriter Effect (สร้างด้วย Vanilla JS) เพื่อดึงดูดความสนใจในส่วน Hero

ดูโค้ด JavaScript Typewriter

4. JavaScript Core Logic (Typewriter Function)

// Typewriter Function (ฟังก์ชันตัวหลัก) function typewriter(targetEl, text, options = {}) { const delay = options.speed || 40; const loop = options.loop || false; const pauseAfter = options.pauseAfter || 1400; let i = 0; let deleting = false; function step() { if (!deleting) { if (i <= text.length) { targetEl.textContent = text.slice(0, i); i++; setTimeout(step, delay); } else if (loop) { setTimeout(() => { deleting = true; i = text.length; step(); }, pauseAfter); } } else { if (i >= 0) { targetEl.textContent = text.slice(0, i); i--; setTimeout(step, Math.max(18, delay * 0.6)); } else { deleting = false; i = 0; setTimeout(step, 220); } } } step(); }
